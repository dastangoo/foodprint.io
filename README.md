# foodprint.io
You are what you eat

## Install

Requires a recent version of [node.js](https://nodejs.org/en/)

```sh
git clone https://github.com/krampstudio/foodprint.io.git
npm install -g grunt-cli
npm install
```

## Preview

```sh
grunt preview
```

## Development

### Build

```sh
grunt build
```

### Development mode

```sh
grunt dev
```
